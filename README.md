## 获得APP list中所有APP的类型（新闻类，健身类......）

1. 在当前项目目录安装puppeteer
  ```sh
   npm i puppeteer
   ```
2. 修改app.py中app list的路径
3. 运行app.py
   ```sh
   python3 app.py
   ```
